# Vue 3 + TypeScript + Vite


## vsCode安装插件

esLint、styleLint

## 安装依赖

pnpm install


## 安装husky

pnpm prepare

## eslint不生效？

pnpm lint

## 项目启动

pnpm start

## 组件格式
    └── 组件名称 xxxx
        ├── components ---子组件
        ├── hooks ---hooks
        ├── index.vue ---默认都为index
        ├── style.scss ---样式文件
        └── type.ts ---ts类型文件
